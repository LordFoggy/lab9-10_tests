Мини-проект команды [Ваша группа/название команды]. Веб-приложение для продажи товаров для сада и фермерства.

##  Стек технологий
* **Backend:** PHP 8+, MySQL
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (Fetch API)

##  Установка и запуск
1. Клонируйте репозиторий: `git clone [ссылка]`
2. Импортируйте дамп базы данных `Green_shop.sql` в вашу СУБД MySQL.
3. Настройте подключение к БД в файле `includes/config.php` (укажите ваш логин и пароль от MySQL).
4. Запустите локальный сервер (Apache/Nginx) и откройте проект в браузере.

##  Основной функционал
* Регистрация и авторизация пользователей (с разделением ролей: клиент/админ).
* Каталог товаров с фильтрацией и сортировкой.
* Асинхронная корзина покупок (без перезагрузки страницы).
* Оформление заказов со списанием остатков со склада.
* Панель администратора для управления товарами, складскими запасами и статусами заказов.
